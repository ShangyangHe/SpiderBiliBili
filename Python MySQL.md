## 1. Mysql概述

+ 关系型数据库管理系统
+ SQL: Structtured Query Language结构化查询语言，用来操作RDBMS
+ 语言分类：
  + DQL|数据查询语言|select
  + DML|操作|insert\update\delete
  + DDL|定义|create\drop
  + TPL|事务处理
  + DCL|控制

## 2.类型与约束

类型：

+ 整型
+ 小数
+ 字符串、可变字符串
+ 枚举

约束：

+ primary key, not null, unique, default
